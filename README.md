# IdeaOasis MVP Starter

## Setup
1. Copy `.env.example` to `.env.local`
2. `npm install`
3. `npm run dev`

## Deploy to Vercel
- Add environment variables from `.env.local` in Vercel project settings.
- Deploy.

## Firestore Rules
Deploy rules in `firestore.rules` to your Firebase project.
